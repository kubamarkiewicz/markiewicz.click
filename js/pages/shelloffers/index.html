Shell Job Offers

IFRAME DEMO: http://en.shelloffers.com/form
Polish website: https://www.shell.pl/careers/praca-na-stacjach-paliw.html
API: http://en.shelloffers.com/form/cms/api
source code: https://github.com/shelloffers/shelloffers-master

This application serves for Shell dealers to publish job offers. It has been published in an iframe on Shell websites in Poland, Slovakia, Czech Relublic, Hungary and Bulgaria.

My role: I was responsible designing the architecture and full-stack development.
technologies: JavaScript (AngularJS, jQuery), CSS (Bootstrap)
API and administration panel in PHP, MySQL (Laravel/October CMS)
