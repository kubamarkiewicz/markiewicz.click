ciudadmigrante.org

URL: http://ciudadmigrante.org
API: http://ciudadmigrante.org/admin/api
source code: https://github.com/kubamarkiewicz/ciudadmigrante

It is an online documentary that contains a lot of audio-visual content (Youtube API) and elements of "social-mapping" (Google Maps API). It is available online and is also installed on a touchscreen in the Museum of History of Immigration in Catalonia. The application is adapted for interaction on touch screen as well as for mobile devices. It is also optimised for search engines and sharing on social platforms (custom pre-render). Application is multilingual.

My role: I was responsible designing the architecture and full-stack development.
technologies: JavaScript (AngularJS, jQuery), CSS (Bootstrap)
API and administration panel in PHP, MySQL (Laravel/October CMS)
