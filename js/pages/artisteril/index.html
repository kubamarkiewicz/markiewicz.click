Artisteril

It is a web application for real-time monitoring and managenet of Automated Guided Vehicles (AGVs) that operate in various manufacturing facilities. Application receives data from the server and sends orders to AGVs via API.

The application was successfully deployed in four factories in automotive industry in Spain and is being developed to further applications.

My role: I was responsible for front-end architecture, development and installation in the factory and designing the API.
technologies: Javascript (AngularJS, AngularJS Material Design, jQuery), HTML5, CSS (Bootstrap)
